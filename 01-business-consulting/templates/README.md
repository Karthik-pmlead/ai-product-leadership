# Templates Library

This folder contains reusable templates for client delivery.

## Categories

### Strategy

- GTM strategy
- Competitive analysis

### Product
- PRD templates
- Roadmaps
- Feature prioritization

### Operations
- SOPs
- Process mapping
- Workflow templates

### Finance
- Forecasting
- ROI analysis
- Budget planning

### PMO
- RAID logs
- Status reports
- Governance trackers

### Investor Relations
- Investor updates
- Pitch summaries
- Board reporting

## Usage Guidelines

1. Duplicate template
2. Rename appropriately
3. Customize content
4. Export final deliverable

## Template Standards

- Use client branding
- Maintain version control
- Keep formatting consistent
- Remove confidential data before reuse
