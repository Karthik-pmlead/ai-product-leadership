- Workshop decks

- `.pptx`
- `.pdf`
- `.key`

---

# Usage Guidelines

## Recommended Workflow

1. Copy the required template
2. Rename using project naming convention
3. Customize content for the client
4. Export deliverable in final format
5. Store finalized versions separately

---

# Naming Convention

```bash
[Client]_[Project]_[Deliverable]_[Version].[ext]
```

### Example

```bash
AcmeCorp_GTMStrategy_FinalDeck_v1.pptx
```

---

# Contribution Standards

When adding templates:
- Keep content reusable
- Remove client-sensitive information
- Use consistent formatting
- Add short descriptions
- Follow folder organization standards
