# Domains (Process Groups)

This folder maps notes to the traditional PMBOK process‑group views.

## Domain list

| Domain                     | Folder link |
|----------------------------|-------------|
| Initiating                 | [initiating](./initiating/) |
| Planning                   | [planning](./planning/) |
| Executing                  | [executing](./executing/) |

| Closing                    | [closing](./closing/) |

## How to use

Each domain folder should contain:
- `overview.md` – key concepts and typical KPIs.
- `process_mapping.md` – links to relevant chapters.
- Optional: `interview_style_questions.md` for scenario‑based revision.
