# Sprint Workflow

## Sprint Lifecycle

Backlog Refinement → Sprint Planning → Daily Standups → Demo → Retro

---

## Workflow Stages

| Stage | Purpose |
|---|---|
| Backlog Ready | Stories refined & estimated |
| Sprint Planning | Capacity allocation |
| In Progress | Active development |
| Review | QA / validation |
| Done | Meets DoD |

---

## Definition of Done

- Code reviewed
- Tests passed
- Acceptance criteria met
- Documentation updated
- Deployed to target environment
