# Model Monitoring

## Monitoring Areas

| Area | Example Metric |
|---|---|
| Accuracy | Prediction quality |
| Drift | Data distribution shift |
| Latency | Inference response time |
| Reliability | Failure rates |

---

## Alert Triggers

- Accuracy drops >5%
- Latency exceeds SLA
- Drift threshold exceeded
