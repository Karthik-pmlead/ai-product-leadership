# Delivery Metrics

| Metric | Definition |
|---|---|
| Velocity | Story points completed |
| Lead Time | Idea → production |
| Cycle Time | Work start → completion |
| Deployment Frequency | Release cadence |
| MTTR | Recovery time after incidents |

---

## Target Benchmarks

- Sprint predictability >90%
- Deployment success >98%
- Velocity variance <10%
