# Playbooks – Security & Compliance Playbooks

This folder (`security-compliance/playbooks/`) contains **actionable playbooks** for product and engineering teams, aligned with:
- GDPR‑style data‑protection,
- ISO‑27001‑style security,
- OWASP‑Top‑10‑style secure‑coding,
- CE‑Marking‑style product‑safety and risk‑assessment.

Each playbook is designed to be:
- Executable in a sprint or launch cycle.  
- Largely language‑agnostic and product‑agnostic (with IoT‑style and web‑style examples).
