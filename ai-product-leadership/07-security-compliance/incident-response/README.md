# Incident Response – Security & Product‑Incidents

This folder (`security-compliance/incident-response/`) contains an **incident‑response framework** for product‑leaders, engineering teams, and security‑leads. It is aligned with:
- GDPR‑style data‑breach‑handling,  
- ISO‑27001‑style information‑security‑incident management,  
- CE‑Marking‑style product‑safety‑and‑near‑miss‑handling.

These materials are designed to be:
- Actionable and executable in real‑time,  
- Reusable across incidents (security, data, product‑safety),  
- Tied back to your risk‑management and audit‑preparation work.
