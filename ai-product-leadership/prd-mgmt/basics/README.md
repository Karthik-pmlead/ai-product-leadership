# 📚 PRD Basics 101

**From requirements gathering → stakeholder approval** | **10 essential steps**

## 📋 PRD Creation Checklist
- [ ] **Problem Statement** (2-3 sentences, business impact)
- [ ] **Success Metrics** (CTR+8%, latency<100ms)
- [ ] **User Personas** (3-5 key segments)
- [ ] **Functional Requirements** (prioritized features)
- [ ] **AI-Specific** (model evals, hallucination tolerance)

## 📖 Core Sections Explained
1. **Executive Summary** - 1 paragraph, C-suite ready
2. **Why Now** - Market timing + competitive edge  
3. **User Stories** - "As a [user], I want [feature] so [benefit]"
4. **Success Criteria** - Model + business KPIs
5. **Risks** - Hallucinations, bias, data drift
