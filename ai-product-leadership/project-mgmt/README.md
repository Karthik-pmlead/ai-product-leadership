## 🚀 AI Project Management Playbook

### **1. Planning (Week 0)**
✅ Define scope (PRD → Epics → Stories)
✅ RACI matrix (50+ engineers aligned)
✅ Risk register (data drift, model staleness)
✅ Capacity planning (DS/Engg/Design)

### **2. Execution (Sprint 0-N)**
✅ Daily standups (15 mins, blocking issues)
✅ Sprint planning (2hr, story points)
✅ Backlog refinement (weekly, tech debt)
✅ Stakeholder sync (bi-weekly demos)

### **3. Launch (Week N+1)**
✅ Canary deployment (1% traffic)
✅ Synthetic monitoring (latency, drift)
✅ Rollback plan (5-min recovery)
✅ Post-mortem (lessons learned)
