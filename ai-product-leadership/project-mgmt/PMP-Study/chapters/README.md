## How to use

Each chapter folder typically contains:
- `summary.md` – one‑page overview.
- `processes.md` – key processes, inputs/outputs, tools.
- `cheatsheet.md` – quick‑review bullets.
- `links.md` – external links (PMI, articles, videos).
