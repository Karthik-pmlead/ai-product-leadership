# Quizzes and MCQs

This folder organizes practice questions for flexible revision.

## Quiz categories

| Type              | Folder link |
|-------------------|-------------|
| By chapter        | [by_chapter](./by_chapter/) |
| By domain         | [by_domain](./by_domain/) |
| Fake exams        | [fake_exams](./fake_exams/) |

## How to use

- Put MCQs in Markdown (e.g., `by_chapter/ch04_mcq.md`).
- Keep answers in `answers/` with clear explanations.
- Reuse these for mock‑exam mode and quick‑drill sessions.
