# Training – Security & Compliance Awareness

This folder (`security-compliance/training/`) contains **security‑and‑compliance training assets** tailored for:
- Product‑leaders and PMs,
- Engineering teams,
- IoT‑and‑hardware‑designers.

These materials are designed to be **lightweight, reusable, and workshop‑ready**, not full‑blown certification courses.
