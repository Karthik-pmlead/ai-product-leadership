# AI Product Metrics

## Core Product Health & Adoption Metrics

---

# 1. Engagement Metrics

* Daily Active Users
* Session depth
* Query frequency

---

# 2. Quality Metrics

* Response relevance score
* Hallucination rate
* Retrieval precision

---

# 3. Adoption Metrics

* Workflow penetration
* Feature utilization
* Retention rate

---

# 4. Experience Metrics

* Latency perception score
* User satisfaction (CSAT)
* Task success rate

---

# 5. Strategic Principle

> A great AI product is defined by usage depth, not usage count.
