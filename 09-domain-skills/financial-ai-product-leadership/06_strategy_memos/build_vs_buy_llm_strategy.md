# Build vs Buy LLM Strategy

## Strategic Memo

---

# 1. Executive Summary

Financial institutions must decide:

> which AI capabilities to build internally vs source externally

---

# 2. Build When

* Core differentiation is required
* Proprietary workflows exist
* Regulatory constraints apply
* Data sensitivity is high

Examples:

* Trading intelligence systems
* Risk models
* Proprietary research systems

---

# 3. Buy When

* Commodity capabilities
* Infrastructure layers
* General-purpose LLM capabilities

Examples:

* Base model APIs
* General summarization
* Standard embeddings

---

# 4. Hybrid Strategy

Most effective approach:

* Buy foundation models
* Build orchestration + workflows + governance layer

---

# 5. Strategic Insight

> Value shifts upward from models to systems.

---

# 6. Final Recommendation

Focus investment on:

* Agent systems
* Workflow integration
* Domain-specific intelligence layers
