# Analytics Frameworks

---

# 🎯 Objective

Define how user behavior and product performance are analyzed.

---

# 🧠 Types of Analytics

| Type | Focus |
|---|---|
| Descriptive | What happened |
| Diagnostic | Why it happened |
| Predictive | What will happen |
| Prescriptive | What should be done |

---

# 📊 Core Models

## Funnel Analysis
Tracks user progression through stages.

## Cohort Analysis
Tracks behavior of user groups over time.

## Behavioral Analytics
Analyzes user interaction patterns.

---

# 🧠 AI Product Analytics

AI systems require:
- model performance tracking
- feedback loop monitoring
- drift detection
- ranking effectiveness

Event → Ingestion → Processing → Aggregation → Insights

- prediction accuracy
