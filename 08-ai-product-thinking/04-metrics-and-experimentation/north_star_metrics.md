# North Star Metrics Framework

---

# 🎯 Objective

Define the single most important metric that reflects product success.

---

# 🧠 What is a North Star Metric?

A North Star Metric represents:
- user value delivered
- long-term business success
- product-market fit signal

---

# 📊 Examples

| Product Type | North Star Metric |
|---|---|
| E-commerce | Conversion rate |
| Recommendation system | CTR / Engagement rate |
| SaaS platform | Daily active users |
| AI Copilot | Task completion rate |

---

# 🧠 AI Product Considerations

AI systems must balance:
- user satisfaction
- model accuracy
- system latency
- business impact
