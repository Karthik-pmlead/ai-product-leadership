# AI Evaluation

---

# Evaluation Objectives

Ensure AI responses are accurate, grounded, safe, and valuable for investment banking workflows.

---

# Offline Metrics

- Precision
- Recall
- F1 Score
- Citation Accuracy
- Grounding Rate

---

# Online Metrics

- User Satisfaction
- Response Latency
- Acceptance Rate
- Task Completion
- Time Saved

---

# Human Evaluation

Review dimensions include:

- Accuracy
- Completeness
- Clarity
- Relevance
- Explainability
- Actionability

---

# Continuous Evaluation

Monitor:

- Prompt performance
- Model drift
- User feedback
- Hallucination trends
- Search quality
