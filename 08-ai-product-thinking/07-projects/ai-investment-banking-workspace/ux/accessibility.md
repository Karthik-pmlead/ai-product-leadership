# Accessibility

---

# Objective

Ensure the Investment Banking Workspace is usable by all employees regardless of ability, device, or assistive technology.

---

# Design Standards

The platform targets WCAG 2.2 AA compliance.

---

# Key Accessibility Features

- Full keyboard navigation
- Screen reader support
- High-contrast themes
- Scalable text
- Accessible forms
- Clear focus indicators
- Descriptive labels
- Captioned multimedia

---

# AI Accessibility

The AI Copilot should:

- Present concise summaries
- Support voice input where available
- Provide alternative representations of complex information
- Avoid relying solely on color or visual cues

---

# Testing Strategy

Accessibility is validated through:

- Automated testing
- Manual audits
- Keyboard-only navigation
- Screen reader testing
- User feedback

---

# Success Metrics

- WCAG compliance score
- Accessibility defect rate
- User satisfaction
- Task completion using assistive technologies
