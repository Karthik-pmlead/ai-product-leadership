# Roadmap & Prioritization

---

# Product Philosophy

Build reusable platform capabilities before feature expansion.

Every release should improve productivity while strengthening the underlying platform.

---

# Prioritization Framework

We use RICE.

Reach

Impact

Confidence

Effort

Priority = (Reach × Impact × Confidence) / Effort

---

# Phase 1 — Foundation (Q1)

Goal

Create the core workspace.

Features

- Enterprise Search
- Company Profiles
- User Authentication
- Workspace Dashboard
- Research Repository
- Notifications

Expected Outcome

Single entry point for banker workflows.

---

# Phase 2 — Productivity (Q2)

Features

- AI Copilot
- Meeting Briefs
- Company Summaries
- Market News
- Alerts
- Saved Searches

Outcome

Reduce research effort.

---

# Phase 3 — Intelligence (Q3)

Features

- Relationship Graph
- Deal Workspace
- Opportunity Tracking
- Knowledge Graph
- AI Recommendations

Outcome

Improve decision quality.

---

# Phase 4 — Automation (Q4)

Features

- Multi-Agent Workflows
- Autonomous Research
- Document Generation
- Predictive Deal Scoring
- Meeting Assistant

Outcome

Increase banker productivity.

Prioritize

- AI
- Knowledge reuse
- Data quality

Delay

- Low-value UI enhancements
- Niche workflow automation
