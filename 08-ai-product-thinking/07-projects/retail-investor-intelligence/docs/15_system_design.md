# System Design

## High-Level Design

- Ingestion Layer (Market + Portfolio Data)
- Processing Layer (Feature Engineering)
- Intelligence Layer (Signal Detection + Ranking)
- Insight Layer (Explanation Generator)
- Presentation Layer (Dashboard)

## Design Principles
- Modularity
- Observability
- Extensibility
- Explainability

## Scalability Path
- Move from CSV → streaming ingestion
- Add distributed processing
- Introduce caching layer
