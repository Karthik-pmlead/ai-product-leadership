# 03 - System Overview

## High-Level Goal

Provide a graph-based intelligence platform capable of:
- monitoring financial relationships
- simulating risk propagation
- generating explainable alerts

---

# Core Components

## Frontend Dashboard
Displays:
- graph visualization
- alerts
- anomalies
- metrics

---

## Backend API
Handles:
- event ingestion
- graph updates
- risk propagation
- WebSocket streaming

---

## Graph Engine
Maintains:
- nodes
- edges
- relationships
- risk scores

---

## Risk Engine
Calculates:
- entity risk updates
- propagation logic
- contagion effects

---

## Anomaly Engine
Detects:
- suspicious concentration
- highly connected risky entities

---

## Explainability Layer
Generates:
- human-readable explanations
- alert reasoning
- propagation rationale

---

# Architecture Style

- event-driven
- graph-oriented
- real-time streaming
- explainable AI system
