# 04 - User Journey

# Step 1 — Open Dashboard

User opens the platform dashboard.

The system displays:
- graph network
- risk metrics
- alerts
- anomaly insights

---

# Step 2 — Observe Financial Network

User views:
- banks
- hedge funds
- exchanges
- traders

connected as a graph.

---

# Step 3 — Trigger Risk Event

User simulates:
- liquidity stress
- exposure event
- suspicious activity

---

# Step 4 — Risk Propagation

System:
- increases node risk
- propagates risk to neighbors
- recalculates metrics

---

# Step 5 — Alert Generation

Platform generates:
- high-risk alerts
- anomaly warnings
- explainability insights

---

# Step 6 — News Intelligence Event

User simulates:
- negative market news
- investigation reports
- sentiment changes

System updates graph risk dynamically.

---

# Step 7 — Continuous Monitoring

Dashboard updates live using:
- WebSockets
- event-driven updates
- real-time graph refreshes
