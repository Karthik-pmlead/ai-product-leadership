# 🧠 Face Recognition Attendance System — System Design (Interview Ready)

## 🎯 Problem Statement

Design a real-time **AI-based biometric attendance system** that:
- Enrolls employees using face images
- Authenticates identity using face recognition
- Prevents spoofing (photos, blur, low-quality inputs)
- Logs attendance with confidence scores
- Provides analytics + metrics dashboard
