# Responsible AI & Security

## Responsible AI Principles

### 1. Privacy by Design
- No raw video stored
- Only derived features (EAR, blink count)

### 2. Transparency
- Rule-based fatigue scoring explained
- No black-box decision making in MVP

### 3. Human-in-the-loop
- Alerts are advisory, not automated control actions

## Bias Considerations
- Eye shape variability across individuals
- Glasses/occlusion may affect detection accuracy

## Safety Considerations
- System is not a medical device
- Alerts are preventive, not authoritative

## Security Controls
- Local processing by default
- No external data transmission in MVP
