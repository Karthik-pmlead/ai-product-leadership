# Product Scope

## In Scope (MVP)
- Real-time driver webcam ingestion via WebRTC
- Face landmark detection (MediaPipe)
- Blink detection (EAR-based)
- Microsleep detection (eye closure duration)
- Fatigue scoring engine (0–100)
- Real-time UI dashboard with trend graph
- Audio alert system for CRITICAL state

## Out of Scope (MVP)
- Multi-driver fleet management
- Cloud deployment (Kubernetes, scaling)
- Deep learning model training pipeline
- Mobile app integration
- Historical database storage

## Future Scope
- Fleet analytics dashboard
- Driver risk profiling over time
- Edge deployment on embedded devices (Jetson)
- Cloud-based monitoring system
