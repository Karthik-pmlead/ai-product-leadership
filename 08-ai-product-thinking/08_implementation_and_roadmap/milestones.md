# Milestones

## Purpose
Track the major delivery checkpoints.

| Milestone | Description |
|---|---|
| Scope locked | Aspect taxonomy and channels agreed. |
| Data ready | Pilot data collected and cleaned. |
| Prototype ready | First working version completed. |
| Evaluation complete | Results reviewed by business and technical teams. |
| Pilot launched | Real users begin using the output. |
| Pilot review | Team decides whether to expand. |
