# Implementation and Roadmap

## Phases
1. define scope.
2. collect sample documents.
3. define target schema.
4. build prototype.
5. validate on test set.
6. launch pilot.
7. improve and expand.

## Milestones
- document type selected.
- schema approved.
- prototype tested.
- pilot launched.
- evaluation complete.
- governance review passed.

## Dependencies
- document access.
- field labels.
- content owners.
- review process.
- downstream system for output.

## Launch plan
Start with one document type and one downstream workflow.

## Adoption plan
Place the extraction output directly into the system or workflow where users already work.
