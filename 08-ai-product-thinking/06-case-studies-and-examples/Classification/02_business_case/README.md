# Business Case

## Business problem
Sales teams waste time manually sorting inbound work, which slows response time and can cause high-value opportunities to be missed.

## ROI model
The expected value comes from faster routing, better prioritization, reduced manual effort, improved lead handling, and more consistent ownership assignment.

## Feasibility
The use case is feasible if the business can define a stable set of categories, has enough historical examples, and can measure routing quality.

## Prioritization
This use case should be prioritized if inbound volume is high, response time matters, and better triage can improve sales productivity or conversion.

## Recommendation
Proceed if the routing problem is frequent, measurable, and costly enough to justify automation.
