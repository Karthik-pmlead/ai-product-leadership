# 05 - AI/ML System Breakdown

# Overview

The platform combines:
- graph analytics
- risk propagation
- anomaly detection
- explainability

---

# 1. Graph Intelligence Layer

Models:
- entities as nodes
- relationships as edges

Enables:
- dependency tracking
- exposure analysis
- contagion modeling

---

# 2. Risk Propagation Engine

When one entity risk increases:
- connected nodes inherit partial risk

This simulates:
- systemic exposure
- counterparty contagion

---

# 3. News Intelligence Layer

Simulated news events:
- modify entity risk
- influence graph behavior

Example:
- investigation
- liquidity concerns
- fraud alerts

---

# 4. Anomaly Detection

Detects:
- highly connected risky entities
- abnormal network concentration

---

# 5. Explainability Layer

Provides:
- human-readable explanations
- alert reasoning
- propagation rationale

---

# Future ML Enhancements

## Possible Upgrades
- Graph Neural Networks (GNNs)
- Temporal graph analysis
- Embedding-based risk scoring
- Predictive contagion modeling
