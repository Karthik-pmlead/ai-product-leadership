# 🎯 Objective

Provide a structured approach for writing effective PRDs.

---

# 🧠 PRD Structure

| Section | Purpose |
|---|---|
| Problem Statement | Define customer problem |
| Goals | Desired outcomes |
| User Personas | Target users |
| Requirements | Functional scope |
| Metrics | Success measurement |
| Risks | Constraints and tradeoffs |

---

# 🚀 AI Product PRD Additions

AI PRDs should include:
- evaluation criteria
- explainability requirements
- model limitations
- human oversight considerations

---

# 📊 Example Requirement

```text
The recommendation engine should update rankings within 2 seconds of user interaction.
```

🎯 Best Practices
- focus on outcomes
- avoid implementation details
- define measurable success
- document assumptions
