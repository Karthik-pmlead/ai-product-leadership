# 🎯 Objective

Evaluate whether features should be built, launched, expanded, or deprecated.

---

# 🧠 Evaluation Dimensions

| Dimension | Question |
|---|---|
| Customer Value | Does it solve a real problem? |
| Business Impact | Does it improve outcomes? |
| Technical Complexity | Is it feasible? |
| Strategic Fit | Does it align with vision? |

---

# 🚀 AI Feature Evaluation

AI features additionally require:
- explainability evaluation
- bias assessment
- monitoring strategy
- retraining considerations

---

# 📊 Example Evaluation Matrix

| Feature | Customer Impact | Complexity | Strategic Fit |
|---|---|---|---|
| AI Copilot | High | High | High |
| Dark Mode | Medium | Low | Medium |

# 🎯 Decision Outcomes

```text
Build
Delay
Experiment
Reject
Deprecate
```
🚀 Best Practices
- validate assumptions
- measure impact

- optimize for learning
