# 📊 Performance Management

> Structured evaluation and feedback system for consistent performance assessment.

---

# 📌 Components

- Goal setting (OKRs)
- Continuous feedback loops
- Mid-cycle reviews
- Annual/quarterly evaluations
- Calibration sessions

---

# 📊 Evaluation Dimensions

| Dimension | Focus |
|---|---|
| Execution | Delivery quality |
| Ownership | Accountability |
| Collaboration | Team impact |
| Growth | Learning velocity |

---

# 🧠 Principles

- No surprises in reviews
- Use data + examples
- Separate behavior from intent
- Consistent calibration across teams

---

# 🎯 Outcome

> Fair, transparent, and consistent performance evaluation.
