# 🧱 Org Design

> Structuring teams, reporting lines, and ownership boundaries for scale.

---

# 📌 Focus Areas

- Team structures (functional, product, platform)
- Reporting hierarchies
- Ownership boundaries
- Manager span of control

---

# 🧭 Common Models

| Model | Use Case |
|---|---|
| Functional | Deep specialization |
| Product-based | End-to-end ownership |
| Platform + Product | Scale efficiency |
| Matrix | Large org coordination |

---

# 🧠 Principles

- Clear ownership > shared ownership
- Minimize cross-team dependencies
- Optimize for decision speed
- Avoid over-layering management

---

# 🎯 Outcome

> Clean, scalable org structure with clear accountability.
