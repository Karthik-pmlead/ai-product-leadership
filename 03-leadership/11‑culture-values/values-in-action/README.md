# ⚙️ Values in Action

> Converting abstract company values into observable behaviors.

---

# 📌 Example Mapping

| Value | Observable Behavior |
|---|---|
| Ownership | Drives issues to resolution |
| Bias for action | Makes decisions with 70% info |
| Transparency | Shares progress + blockers early |
| Collaboration | Engages cross-functional partners early |

---

# 🧭 Implementation

- Define behavioral indicators
- Embed in performance reviews
- Reinforce in retrospectives
- Recognize aligned behavior publicly

---

# ⚠️ Anti-Patterns

- Vague values (“be a leader”)
- Inconsistent enforcement
- Values not tied to evaluation

---

# 🎯 Outcome

> Values become measurable and enforceable behaviors.
