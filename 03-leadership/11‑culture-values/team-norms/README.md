# 🤝 Team Norms

> Explicit working agreements that define how teams operate day-to-day.

---

# 📌 Example Norms

- Response time: <24 hours for blockers
- Meetings: agenda required
- Decisions: documented in decision log
- Code reviews: <48 hours SLA
- Escalations: within 1 working day

---

# 🧭 Norm Categories

| Area | Example |
|---|---|
| Communication | async-first updates |
| Execution | sprint commitments honored |
| Quality | Definition of Done enforced |
| Collaboration | cross-team review required |

---

# ⚠️ Anti-Patterns

- Unwritten expectations
- Inconsistent enforcement
- “Depends on situation” ambiguity

---

# 🎯 Outcome

> Predictable team behavior with minimal friction.
