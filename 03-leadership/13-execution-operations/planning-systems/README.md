# 📅 Planning Systems

> Structured planning cycles that align strategy to execution.

---

# 📊 Planning Levels

| Level | Scope |
|---|---|
| Annual | Strategy + investment themes |
| Quarterly | OKRs + roadmap planning |
| Sprint | Execution planning |
| Weekly | Adjustments & re-planning |

---

# 🧭 Planning Inputs

- OKRs
- Capacity
- Dependencies
- Historical velocity
- Risk signals

---

# ⚠️ Anti-Patterns

- Overplanning without execution feedback
- Static plans that don’t adapt
- No linkage to capacity reality

---

# 🎯 Outcome

> Plans that evolve with execution reality.
