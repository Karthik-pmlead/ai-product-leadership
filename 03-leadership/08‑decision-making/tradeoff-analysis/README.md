# ⚖️ Trade-off Analysis

> Structured approach to evaluating competing priorities and constraints.

---

# 📊 Common Trade-offs

| Trade-off | Example |
|---|---|
| Speed vs Quality | Fast release vs system stability |
| Scope vs Time | Feature cuts to meet deadline |
| Build vs Buy | Internal system vs SaaS tool |
| Short-term vs Long-term | Growth vs platform investment |

---

# 🔄 Decision Flow

Define Options → Evaluate Impact → Score Trade-offs → Decide → Document

---

# 📌 Evaluation Criteria

- Business impact
- Technical risk
- Time to deliver
- Resource cost
- Strategic alignment

---

# 🧠 Principle

> Every decision has a cost — make it visible.

---

# 🎯 Outcome

> Transparent and defensible decisions with clear rationale.
