# 🧩 Decision Frameworks

> Structured models to reduce ambiguity and improve decision quality.

---

# 📊 Key Frameworks

## 1. RAPID Model

| Role | Responsibility |
|---|---|
| Recommend | Propose option |
| Agree | Provide input |
| Perform | Execute decision |
| Input | Consulted stakeholders |
| Decide | Final decision owner |

---

## 2. DACI Model

| Role | Responsibility |
|---|---|
| Driver | Owns execution |
| Approver | Final decision maker |
| Contributors | Provide input |
| Informed | Kept aware |

---

## 3. RICE / Impact Scoring

| Factor | Purpose |
|---|---|
| Reach | Number of users impacted |
| Impact | Degree of effect |
| Confidence | Certainty level |
| Effort | Cost to implement |

---

# 🧠 When to Use

- RAPID → org-level decisions
- DACI → product/engineering decisions
- RICE → prioritization decisions

---

# 🎯 Outcome

> Structured decisions with clear ownership and reduced ambiguity.
