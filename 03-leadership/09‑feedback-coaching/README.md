# 🧠 Core Principles

- Feedback is continuous, not episodic
- Coaching focuses on behavior, not personality
- Clarity builds trust
- High standards + high support = performance
- Growth is a system, not a one-time conversation

# 🎯 Outcome

> Teams that improve continuously while maintaining psychological safety.
