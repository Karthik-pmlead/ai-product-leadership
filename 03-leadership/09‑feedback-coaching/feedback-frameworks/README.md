# 🧩 Feedback Frameworks

> Structured models for delivering clear, actionable, and behavior-focused feedback.

---

# 📊 Key Frameworks

## SBI (Situation – Behavior – Impact)

| Component | Description |
|---|---|
| Situation | Context of the behavior |
| Behavior | What was observed |
| Impact | Effect on team/business |

---

## Start / Stop / Continue

| Type | Purpose |
|---|---|
| Start | New behaviors to adopt |
| Stop | Ineffective behaviors to remove |
| Continue | Effective behaviors to reinforce |

---

## Radical Candor

- Care personally
- Challenge directly

---

# 🧠 Best Practices

- Focus on observable behavior
- Deliver feedback quickly
- Be specific, not general
- Separate intent from impact

---

# 🎯 Outcome

> Clear, actionable feedback that drives behavior change.
