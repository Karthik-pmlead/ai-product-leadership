# 👨‍💻 IC Coaching

> Developing individual contributors for growth, autonomy, and performance excellence.

---

# 📊 Coaching Areas

| Area | Focus |
|---|---|
| Technical Growth | Skill development & depth |
| Execution Quality | Delivery consistency |
| Ownership | Accountability & initiative |
| Communication | Clarity & collaboration |

---

# 🧭 Coaching Model

## 1. Diagnose
- Identify gaps (skill, behavior, ownership)

## 2. Align
- Define expectations clearly

## 3. Develop
- Assign stretch tasks
- Provide guidance

## 4. Review
- Continuous feedback loops

---

# 📌 Coaching Techniques

- Shadowing
- Pair work
- Structured feedback sessions
- Growth plans (30/60/90 days)

---

# 🎯 Outcome

> High-performing ICs with increasing autonomy and impact.
