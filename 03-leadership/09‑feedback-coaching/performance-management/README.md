# 📈 Performance Management

> Structured systems for evaluating, improving, and managing performance fairly and consistently.

---

# 📊 Components

## 1. Goal Setting
- Clear OKRs
- Measurable outcomes

## 2. Continuous Feedback
- Regular check-ins
- Real-time feedback loops

## 3. Performance Reviews
- Quarterly / biannual reviews
- Evidence-based evaluation

---

# 📌 Evaluation Dimensions

| Dimension | Focus |
|---|---|
| Execution | Delivery quality |
| Ownership | Accountability |
| Collaboration | Cross-team effectiveness |
| Growth | Learning & adaptability |

---

# 🧠 Principles

- Evaluate behavior + impact, not personality
- Use data and examples
- Avoid surprises in reviews
- Separate performance from potential

---

# 🎯 Outcome

> Fair, transparent, and growth-oriented performance system.
