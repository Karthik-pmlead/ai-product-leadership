# 📊 Execution Metrics

> Measurement system for engineering and delivery performance.

---

# 📌 Key Metrics

| Metric | Description |
|---|---|
| Velocity | Work completed per sprint |
| Commitment Accuracy | Planned vs delivered work |
| Lead Time | Idea to production time |
| Cycle Time | Work completion speed |
| Throughput | Output per unit time |

---

# 🧠 Principles

- Metrics must drive action
- Focus on trends, not snapshots
- Avoid vanity metrics
- Combine speed + quality indicators

---

# 🎯 Outcome

> Data-driven visibility into execution performance.
