# 🧩 OKR Framework

> Structure for defining meaningful objectives and measurable key results.

---

# 📌 Structure

## Objective
- Clear, inspirational goal

## Key Results
- 2–5 measurable outcomes

---

# 🧠 Rules

- No tasks as KRs
- Must be outcome-based
- Must be measurable
- Limited to focus priorities

---

# 🎯 Outcome

> Clear definition of success.
