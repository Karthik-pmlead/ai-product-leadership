# 🗓️ Meetings System

> Structured approach to making meetings efficient, purposeful, and decision-driven.

---

# 📌 Meeting Types

| Type | Purpose |
|---|---|
| Decision | Make binding decisions |
| Sync | Align progress |
| Planning | Define execution |
| Review | Evaluate outcomes |

---

# 🧭 Meeting Structure

1. Objective (why meeting exists)
2. Pre-read (mandatory)
3. Discussion (time-boxed)
4. Decision (explicit)
5. Action items (owners + deadlines)

---

# 🧠 Principles

- No agenda = no meeting
- Every meeting must end with decisions or actions
- Pre-read reduces meeting time by 50%
- Time-box everything

---

# 🎯 Outcome

> Efficient meetings with clear outcomes.
