# 💬 Async Communication

> Communication optimized for distributed teams and minimal real-time dependency.

---

# 📌 Channels

- Slack updates
- Email threads
- Status dashboards
- Written updates

---

# 🧭 Patterns

## 1. Status Update Format
- What changed
- Progress
- Blockers
- Next steps

---

## 2. Decision Request Format
- Context
- Options
- Recommendation
- Deadline

# 🧠 Principles

- Assume asynchronous reading
- Be structured and scannable
- Avoid long unstructured messages
- Reduce back-and-forth cycles

# 🎯 Outcome

> Faster alignment without meetings.
