# ✍️ Written Communication

> Structured writing for clarity, alignment, and decision-making.

---

# 📌 Focus Areas

- PRDs and specs
- Design documents
- Decision memos
- Status updates
- Technical proposals

---

# 🧭 Writing Structure

Context → Problem → Options → Recommendation → Impact

---

# 🧠 Principles

- Short and structured
- Data-backed reasoning
- One idea per section
- No ambiguity in decisions

---

# 🎯 Outcome

> Clear documentation that drives action.
