# ⚖️ Conflict & Influence

> Managing disagreement, navigating ambiguity, and influencing outcomes without authority.

---

# 🎯 Purpose

This section defines how leaders:
- Resolve conflict constructively
- Influence cross-functional decisions
- Handle ambiguity and competing priorities
- Align stakeholders without formal authority

# 🧠 Core Principles

- Conflict is data, not disruption
- Influence is earned, not assigned
- Clarity reduces resistance
- Decisions require trade-offs, not consensus
- Alignment > agreement

# 🎯 Outcome

> Faster decisions, reduced friction, and aligned execution across teams.
