# 🤝 Stakeholder Management

> Identifying, mapping, and managing expectations across stakeholders to ensure alignment and execution success.

---

# 📊 Stakeholder Types

| Type | Role |
|---|---|
| Executives | Strategy & funding |
| Product | Prioritization |
| Engineering | Execution |
| Design | UX alignment |
| Data/AI | Insights & models |

---

# 🧭 Engagement Model

Identify → Map → Prioritize → Engage → Align → Maintain

---

# 📌 Stakeholder Matrix

| Level | Behavior |
|---|---|
| Unaware | Needs communication |
| Neutral | Needs alignment |
| Supportive | Needs engagement |
| Leading | Drives decisions |

---

# 🧠 Best Practices

- Early alignment reduces resistance
- Regular updates prevent surprises
- Tailor communication by stakeholder type
- Document decisions clearly

---

# 🎯 Outcome

> Predictable stakeholder alignment and reduced execution friction.
