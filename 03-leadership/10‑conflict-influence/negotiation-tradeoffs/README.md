# ⚖️ Negotiation & Trade-offs

> Making and communicating decisions where not all priorities can be satisfied.

---

# 📊 Common Trade-offs

| Trade-off | Example |
|---|---|
| Speed vs Quality | Fast release vs stable system |
| Scope vs Timeline | Feature reduction for deadline |
| Build vs Buy | Internal vs external solutions |
| Innovation vs Stability | New tech vs proven systems |

---

# 🔄 Decision Flow

Context → Options → Trade-offs → Decision → Communication → Alignment

---

# 📌 Principles

- Every decision has a cost
- Make trade-offs explicit
- Document rationale
- Avoid hidden compromises

---

# 🎯 Outcome

> Clear, defensible decisions with stakeholder alignment.
