# 🧩 Conflict Resolution

> Structured approach to identifying, addressing, and resolving conflicts in teams and cross-functional environments.

---

# 🧭 Conflict Types

| Type | Example |
|---|---|
| Priority conflict | Competing roadmap items |
| Resource conflict | Shared engineering bandwidth |
| Technical conflict | Architecture disagreements |
| Process conflict | Delivery or workflow issues |
| Interpersonal conflict | Communication breakdowns |

---

# 🔄 Resolution Flow

Identify → Clarify → Align → Decide → Document → Follow-up

---

# 📌 Techniques

- Data-driven discussion
- 1:1 resolution before escalation
- Written alignment docs
- Neutral facilitation
- Decision logging

---

# ⚠️ Anti-Patterns

- Avoiding conflict
- Political escalation too early
- Unstructured debates
- No documented resolution

---

# 🎯 Outcome

> Faster resolution with reduced emotional friction.
