# Neural Networks Knowledge Base (NN)

Welcome to the **Neural Networks Knowledge Base**, a curated collection of deep learning fundamentals, architectures, and production techniques. Covers forward/backward propagation, TensorFlow, optimization, and specialized networks.

This repository is useful for:
- Understanding core deep learning mechanics from scratch implementations.
- Building and tuning production neural networks with TensorFlow/Keras.
- Mastering optimization, regularization, and architecture design patterns.
