# Technical Depth

The **Technical** folder showcases my hands‑on depth in data science, ML, and engineering that underpins product and leadership decisions.  

It acts as a central hub to all my external technical repositories, organized by capability (SQL, Tableau, Machine Learning, NLP/RecSys, Data Engineering, Data Modeling, and System Design). Hiring managers and partners can use this section to quickly understand the scope of my technical work and follow links to full projects, notebooks, and production‑ready artifacts.

| Folder | Purpose / Content |
|--------|-------------------|
| `DataScience/SQL` | SQL projects, data cleaning, and complex queries; links to your external SQL repo. |
| `DataScience/Tableau` | Interactive dashboards and visualizations; links to your Tableau projects repo. |
| `DataScience/MachineLearning` | Supervised/unsupervised ML, model evaluation, and case studies; links to your ML repo. |
| `DataScience/NLP` | NLP and recommender‑system projects (e.g., RecSys, embeddings, ranking); links to relevant repo. |
| `DataEngineering` | Data pipelines, ingestion, orchestration, and batch/stream processing. |
| `DataModeling` | Schema design, star‑schema, data marts, and feature‑store layouts. |
| `SystemDesign` | Architecture sketches, trade‑offs, and system design notes for AI/ML products. |
