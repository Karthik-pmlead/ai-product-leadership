# Blue Ocean Strategy

## Overview

Blue Ocean Strategy is a strategic framework focused on:
- creating uncontested markets
- reducing competition
- generating new demand

Instead of competing in crowded markets (“Red Oceans”),
organizations create new value spaces (“Blue Oceans”).

Created by:
- W. Chan Kim
- Renée Mauborgne

---

# Core Idea

Avoid competing directly.

Instead:
- redefine value
- innovate business models
- create new categories

---

# Red Ocean vs Blue Ocean

| Red Ocean | Blue Ocean |
|---|---|
| Compete in existing markets | Create new markets |
| Beat competitors | Make competition irrelevant |
| Existing demand | New demand |
| Incremental innovation | Value innovation |

---

# Four Actions Framework

| Action | Question |
|---|---|
| Eliminate | What can be removed? |
| Reduce | What can be minimized? |
| Raise | What should exceed standards? |
| Create | What new value can be introduced? |

---

# Example

## Cirque du Soleil

Created a new category by combining:
- circus
- theater
- storytelling

While eliminating:
- animal acts
- low-cost positioning

---

# Best Used When

- Market saturation
- Innovation strategy
- Category creation
- Product differentiation

---

# Advantages

- Encourages innovation
- Reduces price competition
- Creates strategic differentiation

---

# Limitations

- Difficult execution
- Blue oceans eventually become competitive
- Requires strong innovation capability

---

# Best Practices

- Focus on customer value innovation
- Challenge industry assumptions
- Explore non-customers
