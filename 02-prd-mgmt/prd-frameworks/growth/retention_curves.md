# Retention Curves

## Overview

Retention Curves visualize how many users continue using a product over time.

They help teams understand:
- user stickiness
- churn behavior
- product-market fit
- engagement quality

Retention analysis is critical for:
- SaaS
- consumer apps
- subscription businesses

---

# Core Idea

Strong products retain users consistently over time.

The curve shows:
- how quickly users drop off
- whether retention stabilizes

---

---

# Key Patterns

| Pattern | Meaning |
|---|---|
| Sharp drop then flattening | Healthy long-term retention |
| Continuous decline | Weak retention |
| Flat curve | Strong habit formation |

---

# Common Retention Types

| Type | Description |
|---|---|
| Day N Retention | Return on specific day |
| Rolling Retention | Active after a period |
| Cohort Retention | Group-based retention over time |
