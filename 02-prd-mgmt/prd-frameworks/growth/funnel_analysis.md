# Funnel Analysis

## Overview

Funnel Analysis measures how users progress through a sequence of steps toward a desired outcome.

It helps identify:
- drop-off points
- friction
- conversion opportunities

Funnels are commonly used in:
- growth
- product analytics
- marketing
- onboarding optimization

---

# Example Funnel

```text
Visit Site
   ↓
Signup
   ↓
Complete Onboarding
   ↓
Activate Feature
   ↓
Purchase
```

---

# Core Metrics

| Metric | Description |
|---|---|
| Conversion Rate | % moving to next step |
| Drop-off Rate | % leaving funnel |
| Completion Rate | % completing entire flow |
| Time-to-Conversion | Time between steps |

---

# Example

| Funnel Step | Users | Conversion |
|---|---|---|
| Website Visit | 10,000 | — |
| Signup | 2,500 | 25% |
| Onboarding Complete | 1,600 | 64% |
| Paid Conversion | 300 | 18% |

---

# Advantages

- Identifies friction clearly
- Improves conversion optimization
- Supports experimentation
- Easy visualization

---

# Limitations

- Real user journeys are non-linear
- Doesn't explain *why* drop-offs happen
- Over-focus on conversion may hurt UX

---

# Best Practices

- Analyze cohorts separately
- Combine quantitative + qualitative insights
- Focus on high-impact drop-offs
- Track time between steps
