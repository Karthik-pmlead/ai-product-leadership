# Behavioral Segmentation

## Overview

Behavioral Segmentation groups users based on:
- actions
- usage patterns
- engagement behavior
- purchasing habits

Instead of demographics, it focuses on:
- what users do

---

# Common Segmentation Types

| Type | Description |
|---|---|
| Usage Frequency | Daily vs occasional users |
| Purchase Behavior | High vs low spenders |
| Engagement Level | Active vs dormant users |
| Feature Adoption | Which features users use |
| Loyalty | Retained vs churn-prone |

---

# Examples

| Segment | Behavior |
|---|---|
| Power Users | Daily active users |
| Explorers | Try many features |
| Passive Users | Rare engagement |
| Churn Risk Users | Declining activity |

---

# Best Used When

- Personalization
- Growth optimization
- Retention strategies
- CRM campaigns
- Feature adoption analysis

---

# Advantages

- Highly actionable
- Better than pure demographics
- Enables personalization
- Improves targeting

---

# Limitations

- Requires analytics infrastructure
- User behavior changes frequently
- Data quality is critical
