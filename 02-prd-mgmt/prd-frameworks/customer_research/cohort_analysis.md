# Cohort Analysis

## Overview

Cohort Analysis groups users based on shared characteristics or experiences over time.

It helps analyze:
- retention
- engagement
- churn
- monetization trends

A cohort is typically:
- users who started during the same time period

---

# Common Cohort Types

| Cohort Type | Example |
|---|---|
| Acquisition Cohort | Users joined in January |
| Behavioral Cohort | Users who used Feature X |
| Subscription Cohort | Users on premium plan |

---

# Retention Example

| Cohort | Week 1 | Week 2 | Week 4 |
|---|---|---|---|
| Jan Users | 100% | 72% | 40% |
| Feb Users | 100% | 80% | 55% |

---

# Best Used When

- SaaS analytics
- Retention analysis
- Subscription products
- Product-market fit analysis
- Growth optimization

---

# Key Insights

Cohort analysis helps answer:
- Are users retaining better over time?
- Did onboarding improve retention?
- Which user groups churn fastest?

---

# Advantages

- Reveals long-term trends
- Better than aggregate metrics
- Helps identify churn drivers
- Useful for experimentation

---

# Limitations

- Requires clean analytics
- Can become overly complex
- Small cohorts may mislead

---

# Example

A new onboarding flow increases:
- Week 4 retention from 35% → 52%

This suggests onboarding improvements worked.
