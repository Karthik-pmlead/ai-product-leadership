# OODA Loop

## Overview

OODA Loop is a rapid decision-making framework created by:
- John Boyd

OODA stands for:
- Observe
- Orient
- Decide
- Act

It emphasizes:
- adaptability
- speed
- continuous learning

---

# OODA Cycle

| Step | Purpose |
|---|---|
| Observe | Gather information |
| Orient | Analyze context |
| Decide | Choose action |
| Act | Execute quickly |

---

# Flow

```text
Observe → Orient → Decide → Act → Repeat
```

---

# Core Principle

Organizations that:
- learn faster
- adapt faster
- respond faster

gain strategic advantage.

---

# Example

## Incident Response

| Step | Example |
|---|---|
| Observe | Detect outage |
| Orient | Analyze root issue |
| Decide | Roll back deployment |
| Act | Restore service |
