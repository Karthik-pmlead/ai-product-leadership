# 🎨 Double Diamond Framework

## 📌 What is it?

The **Double Diamond** is a design thinking framework used to solve problems through a structured process of **diverging (exploring)** and **converging (deciding)**.

It helps teams move from **problem discovery → solution delivery**.

- Design the right thing, Design things right.
- Validated problem statement: Problem -> Insight into problem -> Scope down the focus
- Verfied solutions: validate problem statement -> Potential solutions -> Solutions that work & receive feedback

---

## 🧠 The 4 Stages

- 🔍 Discover  
- 🎯 Define  
- 💡 Develop  
- 🚀 Deliver  

---

## ⏰ When to Use Double Diamond

Use it when:

- You are solving **ambiguous or unclear problems**
- You need **deep user research**
- You are designing a **new product or feature**
- You want to reduce **guesswork in decision-making**
- You are working on **UX / product discovery**

👉 Best for: **problem-solving + product design**

---

## 📍 Where to Use It

- Product discovery phase  
- UX research  
- Feature design  
- Problem validation  
- Innovation workshops  

---

## ⚙️ Double Diamond Flow

| Phase     | Type        | Meaning | PM / Design Action |
|-----------|------------|--------|---------------------|
| 🔍 Discover | Diverge | Explore problem space | User research, interviews, data analysis |
| 🎯 Define   | Converge | Narrow down problem | Synthesize insights, define core problem |
| 💡 Develop  | Diverge | Explore solutions | Brainstorming, prototyping, ideation |
| 🚀 Deliver  | Converge | Final solution | Build, test, launch |

---

## 🧩 Simple Mental Model

- 🔍 Discover → “What is the real problem?”
- 🎯 Define → “What exactly should we solve?”
- 💡 Develop → “What are possible solutions?”
- 🚀 Deliver → “What should we ship?”

---

## 🍔 Example (Food Delivery App)

| Phase     | Example |
|-----------|--------|
| Discover  | Users complain about late deliveries |
| Define    | Core problem: “Unreliable delivery ETA” |
| Develop   | Ideas: live tracking, better routing, incentives |
| Deliver   | Ship real-time delivery tracking feature |

---

## 🔥 PM Insight

> The Double Diamond prevents teams from jumping to solutions too early.

It forces:
- Proper **problem understanding**
- Structured **idea exploration**
- Validated **solution delivery**

---

## 🧠 Why it matters

- Avoids building wrong features
- Improves product-market fit
- Aligns UX, design, and engineering early
- Encourages user-centered thinking
