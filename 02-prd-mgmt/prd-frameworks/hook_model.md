# Hook Model

## Overview

The Hook Model is a behavioral design framework created by:
- Nir Eyal

It explains how products build:
- user habits
- recurring engagement
- behavioral loops

Popular in:
- consumer apps
- social products
- gamified systems

---

# The 4-Step Hook Cycle

| Step | Purpose |
|---|---|
| Trigger | Initiates behavior |
| Action | Simplest user action |
| Variable Reward | Creates engagement |
| Investment | Increases future return probability |

---

# Hook Cycle

```text
Trigger
   ↓
Action
   ↓
Variable Reward
   ↓
Investment
   ↓
Repeat
```

---

# Step Breakdown

## Trigger

Types:
- external (notifications)
- internal (emotion, boredom)

Example:
- Instagram notification

---

## Action

The simplest behavior.

Example:
- opening app
- scrolling feed

---

## Variable Reward

Unpredictable rewards create stronger engagement.

Examples:
- likes
- new content
- social feedback

---

## Investment

User contributes effort/data.

Examples:
- profile creation
- followers
- saved preferences

This increases switching costs.

---

# Best Used When

- Consumer products
- Social platforms
- Habit-forming apps
- Engagement optimization

---

# Advantages

- Strong engagement model
- Explains behavioral retention
- Useful for product design

---

# Limitations

- Ethical concerns around addiction
- Doesn't guarantee product value
- Habit ≠ satisfaction
