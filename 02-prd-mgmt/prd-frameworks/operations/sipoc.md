# SIPOC Diagram

## Overview

SIPOC is a high-level process mapping framework used to understand process boundaries and stakeholders.

SIPOC stands for:
- Suppliers
- Inputs
- Process
- Outputs
- Customers

It is commonly used in:
- Six Sigma
- process improvement
- operational analysis

---

# SIPOC Structure

| Element | Description |
|---|---|
| Suppliers | Who provides inputs |
| Inputs | Resources/data/materials |
| Process | Key process steps |
| Outputs | Process outcomes |
| Customers | Who receives outputs |

---

# Example

## E-Commerce Checkout

| SIPOC Element | Example |
|---|---|

# Best Used When

- Process mapping
- Operational reviews

---

# Advantages

- Simple high-level process view
- Clarifies stakeholders
- Improves process understanding
- Useful for alignment workshops

---

# Limitations

- High-level only
- Doesn't capture detailed workflows
- Limited root-cause analysis capability

---

# Best Practices

- Keep process steps concise
- Focus on process boundaries
- Identify customer expectations
- Use collaboratively
