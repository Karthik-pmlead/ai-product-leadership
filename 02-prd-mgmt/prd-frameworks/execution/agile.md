# Agile

## Overview

Agile is an iterative approach to product development focused on:
- adaptability
- collaboration
- continuous delivery
- customer feedback

Agile prioritizes:
- responding to change
over rigid long-term planning.

---

# Agile Manifesto Values

| Agile Values |
|---|
| Individuals & interactions over processes & tools |
| Working software over comprehensive documentation |
| Customer collaboration over contract negotiation |
| Responding to change over following a plan |

---

# Core Principles

- Incremental delivery
- Continuous feedback
- Cross-functional collaboration
- Iterative improvement
- Customer-centric development

---

# Typical Agile Cycle

```text
Plan
  ↓
Build
  ↓
Test
  ↓
Release
  ↓
Learn
  ↓
Repeat
```

# Advantages

- Faster feedback cycles
- Increased adaptability
- Better customer alignment
- Continuous improvement

---

# Limitations

- Requires organizational discipline
- Poor execution creates chaos
- Hard for fixed-scope environments

# Best Practices

- Deliver incrementally
- Prioritize customer feedback
- Keep teams cross-functional
- Embrace retrospectives
