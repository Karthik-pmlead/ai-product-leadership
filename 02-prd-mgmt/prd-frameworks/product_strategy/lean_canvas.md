# Lean Canvas

## Overview

Lean Canvas is a startup-focused adaptation of the Business Model Canvas.

Created by:
- Ash Maurya

It emphasizes:
- risk
- uncertainty
- problem validation
- MVP thinking

---

# Lean Canvas Sections

| Section | Description |
|---|---|
| Problem | Core customer problems |
| Customer Segments | Target users |
| Unique Value Proposition | Why product matters |
| Solution | Proposed solution |
| Channels | Acquisition/distribution |
| Revenue Streams | Monetization |
| Cost Structure | Major expenses |
| Key Metrics | Success metrics |
| Unfair Advantage | Defensible advantage |

---

# Core Difference from BMC

Lean Canvas focuses more on:
- startups
- experimentation
- problem-solution fit

---

# Best Used When

- MVP planning
- Startup ideation
- Early-stage validation
- Product discovery

---

# Advantages

- Startup-friendly
- Encourages hypothesis thinking
- Fast to create
- Supports experimentation

---

# Limitations

- Simplified business view
- Less useful for mature enterprises
