# Business Model Canvas (BMC)

## Overview

The Business Model Canvas is a strategic framework used to visualize how a business creates, delivers, and captures value.

Created by:
- Alexander Osterwalder

It provides a one-page overview of a business model.

---

# The 9 Building Blocks

| Block | Description |
|---|---|
| Customer Segments | Target customers |
| Value Propositions | Customer value delivered |
| Channels | Delivery mechanisms |
| Customer Relationships | Engagement approach |
| Revenue Streams | How money is earned |
| Key Resources | Critical assets |
| Key Activities | Core operations |
| Key Partnerships | External collaborators |
| Cost Structure | Major costs |

---

# Visualization

```text
Key Partners | Key Activities | Value Proposition | Customer Relationships | Customer Segments
Key Resources |                |                   | Channels               |

Cost Structure | Revenue Streams
```

# Limitations

- High-level abstraction
- Doesn't capture execution complexity
- Needs frequent updates
