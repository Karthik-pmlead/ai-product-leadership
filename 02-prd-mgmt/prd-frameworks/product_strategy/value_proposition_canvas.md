# Value Proposition Canvas

## Overview

The Value Proposition Canvas helps teams align products with customer needs.

It connects:
- customer jobs
- pains
- gains

with:
- product value
- pain relievers
- gain creators

Created by:
- Alexander Osterwalder

---

# Two Main Sections

## Customer Profile

| Area | Description |
|---|---|
| Jobs | Tasks users want done |
| Pains | Frustrations/problems |
| Gains | Desired outcomes |

---

## Value Map

| Area | Description |
|---|---|
| Products & Services | What is offered |
| Pain Relievers | How pains are reduced |
| Gain Creators | How benefits are created |

---

# Core Goal

Achieve:
> Product–Market Fit

---

# Example

## Customer Job
Organize meeting notes quickly.

## Pain
Manual summarization is time-consuming.

## Gain
Fast searchable summaries.

## Solution
AI-generated structured notes.
