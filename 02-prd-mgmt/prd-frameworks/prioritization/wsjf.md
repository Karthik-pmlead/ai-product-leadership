# Weighted Shortest Job First (WSJF)

## Overview

WSJF is a prioritization framework used in SAFe (Scaled Agile Framework) to maximize economic value delivery.

It prioritizes initiatives with:
- high value
- high urgency
- low effort

# Components

| Component | Description |
|---|---|
| User-Business Value | Customer/business impact |
| Time Criticality | Urgency |
| Risk Reduction | Risk mitigation or opportunity enablement |
| Job Size | Effort/complexity |

---

# Best Used When

- SAFe organizations
- Enterprise backlogs
- Large portfolio prioritization
- Multi-team planning

---

# Advantages

- Economic prioritization
- Balances value vs effort
- Scales well for enterprises
- Supports Agile planning

---

# Limitations

- Complex scoring
- Requires estimation maturity
- May favor small tasks excessively

---

# Example

| Initiative | CoD | Job Size | WSJF |
|---|---|---|---|
| Security patch | 20 | 2 | 10 |
| Analytics dashboard | 18 | 6 | 3 |
| UI redesign | 15 | 8 | 1.8 |

Higher score = higher priority.
