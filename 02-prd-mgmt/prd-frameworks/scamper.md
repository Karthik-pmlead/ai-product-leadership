# 💡 SCAMPER Framework

## 📌 What is it?

**SCAMPER** is a creative ideation framework used to generate new ideas or improve existing products by asking structured questions.

It helps PMs and teams **think beyond obvious solutions**.

---

## 🧠 SCAMPER Meaning

Each letter represents a different way to modify or rethink a product:

- 🔄 S → Substitute  
- 🔀 C → Combine  
- ✏️ A → Adapt  
- 🔧 M → Modify (or Magnify/Minify)  
- 🔁 P → Put to another use  
- ❌ E → Eliminate  
- 🔄 R → Reverse (or Rearrange)  

---

## ⏰ When to Use SCAMPER

Use SCAMPER when:

- You are stuck in **idea generation**
- You want to improve an existing product
- You are exploring **feature enhancements**
- You need **creative brainstorming in workshops**
- You want to find **alternative solutions quickly**

👉 Best for: **ideation + product innovation**

---

## 📍 Where to Use It

- Product brainstorming sessions  
- UX/design workshops  
- Feature improvement discussions  
- Startup idea exploration  
- Innovation labs  

---

## ⚙️ SCAMPER Table

| Letter | Meaning | Question to Ask | Example (Food Delivery App) |
|--------|--------|------------------|------------------------------|
| 🔄 S | Substitute | What can we replace? | Replace human support with AI chatbot |
| 🔀 C | Combine | What can we merge? | Combine food delivery + grocery delivery |
| ✏️ A | Adapt | What can we borrow from others? | Adapt Uber-style real-time tracking |
| 🔧 M | Modify | What can we change or scale? | Make delivery faster (15 min delivery) |
| 🔁 P | Put to another use | Can this feature serve another purpose? | Use order history for recommendations |
| ❌ E | Eliminate | What can we remove? | Remove unnecessary checkout steps |
| 🔄 R | Reverse | What if we flip the process? | Customers bid for faster delivery slots |

---

## 🧠 Simple Mental Model

- 🔄 Change something (Substitute, Modify)
- 🔀 Combine ideas
- ✏️ Adapt from others
- 🔁 Reuse differently
- ❌ Remove friction
- 🔄 Flip the system

---

## 🍔 Example (Food Delivery App Insight)

Using SCAMPER:

- Combine → food + grocery delivery
- Eliminate → long checkout flow
- Adapt → Amazon-style recommendations
- Reverse → let users set delivery priority price
