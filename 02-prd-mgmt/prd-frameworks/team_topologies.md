# Team Topologies

## Overview

Team Topologies is an organizational design framework focused on optimizing:
- team structure
- communication
- software delivery flow

Created by:
- Matthew Skelton
- Manuel Pais

---

# Core Idea

Organizations should design teams to:
- reduce cognitive load
- improve delivery flow
- enable fast software evolution

---

# The 4 Team Types

| Team Type | Purpose |
|---|---|
| Stream-Aligned Team | Owns customer/business flow |
| Platform Team | Provides internal platform services |
| Enabling Team | Helps teams adopt practices |
| Complicated Subsystem Team | Handles specialized complexity |

---

# Interaction Modes

| Mode | Meaning |
|---|---|
| Collaboration | Close joint work |
| X-as-a-Service | Service consumption |
| Facilitating | Temporary guidance/support |
