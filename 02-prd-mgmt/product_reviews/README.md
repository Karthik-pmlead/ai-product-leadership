# Suggested PRD Review Format

```markdown
# Product / Feature
# Problem Statement
# Target Users
# Goals & Metrics
# Scope
# UX / Workflow
# Technical Considerations
# Risks & Dependencies
# PM Critique
# Improvements
# Final Assessment
```
