# PM Workflow Workspace

A living workspace for day-to-day product management.

This directory contains ongoing product work across:
- customer insights
- discovery
- product definition
- planning
- communication
- execution

---

# Operating Principles

- Keep docs lightweight
- Prefer many small docs
- One topic per file
- Capture decisions continuously
- Optimize for discoverability
- Treat this workspace as a living system

---

# Suggested Daily Workflow

```text
1. Capture customer insights
2. Explore opportunities
3. Define product direction
4. Prioritize and plan
5. Communicate decisions
6. Execute and iterate
```
