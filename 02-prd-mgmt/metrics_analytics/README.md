## How to use this repo

- Use this repo as a metric reference base for product, business, and analytics work.
- Each file groups related metrics under one category for easier browsing.
- Add new metrics in the most relevant category first, then link them here.
- Keep definitions short, practical, and decision-oriented.

## Taxonomy

- Transaction products: metrics tied to purchases, payments, bookings, orders, and conversions.
- Attention products: metrics tied to content consumption, engagement, retention, and reach.
- Productivity products: metrics tied to task completion, throughput, efficiency, and workflow health.
- CRM and segmentation: metrics tied to customer lifecycle, account health, and cohorts.
- Acquisition and marketing: metrics tied to channels, campaigns, and paid growth.
- Revenue and finance: metrics tied to monetization, margins, and business health.
- Product quality and support: metrics tied to reliability, satisfaction, and service quality.

## Conventions

- Prefer one metric per row.
- Add formulas later if needed.
- Use consistent names across files.
- Keep metric descriptions concise and reusable.
