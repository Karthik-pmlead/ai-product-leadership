# FAQ

## Overview

This document captures common questions and objections related to the launch.

## Common questions

### What is Guided Onboarding?

Guided Onboarding is a step-by-step experience that helps new users complete setup and reach value faster.

### Who can use it?

It is available to new self-serve users during the initial rollout.

### Will this change my existing workflow?

No. Existing users can continue using the product as before.

### Why am I seeing reminders?

If you pause during setup, you may receive a reminder to help you continue.

### Can I turn off the reminders?

Yes, reminder frequency is controlled and can be adjusted based on user behavior and preferences.

### What if I still need help?

You can contact support or read the user guide for more detail.
