# Marketing Guide

## Overview

This document gives marketing teams the product message, value proposition, audience focus, and campaign guidance needed for launch.

## Product details

| Field | Value |
|---|---|
| Product | Guided onboarding and lifecycle nudges |
| Owner | Product Marketing |
| Launch date | 2026-06-12 |
| Audience | Marketing, CRM, content, growth |

## Core message

| Item | Details |
|---|---|
| Message | Help users reach value faster with guided onboarding. |
| Value proposition | Reduce friction, improve activation, and increase retention. |
| Proof point | Early testing shows higher setup completion and lower drop-off. |

## Audience

| Segment | What they care about |
|---|---|
| New users | Getting started quickly |
| Returning inactive users | A simple way to come back |
| Internal teams | Clear messaging and consistent positioning |

## Messaging pillars

| Pillar | Supporting idea |
|---|---|
| Faster time to value | Users do not have to guess the next step. |
| Less confusion | Tooltips and checklists reduce friction. |
| Better retention | Nudges help users stay engaged. |

## Approved phrases

| Use this | Avoid this |
|---|---|
| Guided onboarding | Magic onboarding |
| Faster time to value | Instant success |
| Contextual help | Smart AI magic |

## Campaign guidance

| Channel | Recommended angle |
|---|---|
| Email | Focus on simplicity and getting started faster. |
| Blog | Explain the user problem and solution. |
| In-app messaging | Highlight the next best action. |
| Social | Show the benefit in one short sentence. |

## Notes

- Keep messaging consistent across channels.
- Use the approved value proposition.
- Update the guide when positioning changes.
