# Enablement Templates

Reusable templates for customer, support, and marketing enablement.

## Navigation

| Template | Purpose | File |
|---|---|---|
| Support Guide | Help support teams resolve common issues and answer user questions. | [support-guide.md](./support-guide.md) |
| User Guide | Help end users understand how to use the product or feature. | [user-guide.md](./user-guide.md) |
| Marketing Guide | Provide messaging, value props, proof points, and campaign guidance. | [marketing-guide.md](./marketing-guide.md) |

## What’s inside

| File | Best for |
|---|---|
| `support-guide.md` | Support team troubleshooting and response consistency. |
| `user-guide.md` | Customer self-service and task completion. |
| `marketing-guide.md` | Campaign messaging and product positioning. |

## Suggested use order

1. Use `user-guide.md` to explain how the product works for customers.
2. Use `support-guide.md` to help support answer questions consistently.
3. Use `marketing-guide.md` to align campaigns and launch messaging.

## Notes

- Keep guides practical and task-oriented.
- Tailor the language to the audience.
- Update the docs whenever the product changes.
