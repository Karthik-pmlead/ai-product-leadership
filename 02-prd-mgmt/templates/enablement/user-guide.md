## Getting started

| Step | What to do |
|---|---|
| 1 | Log in to your account. |
| 2 | Open the onboarding checklist. |
| 3 | Complete the first setup step. |
| 4 | Review tooltips if you need help. |
| 5 | Finish setup and continue using the product. |

## Key features

| Feature | What it does |
|---|---|
| Onboarding checklist | Shows the most important next steps. |
| Tooltips | Explains key screens and actions. |
| Reminder nudges | Helps you return if you pause during setup. |

## Common tasks

### How do I start setup?

Open the checklist from your dashboard and complete the first item to begin.

### How do I get help?

Use the tooltip guidance in the product or contact support if you still need assistance.

### What if I stop midway?

You may receive a reminder to continue from where you left off.

## Tips

- Complete the checklist in order.
- Keep notifications on if you want reminders.
- Use the help text if a step is unclear.

## Notes
