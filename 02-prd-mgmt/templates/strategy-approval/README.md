# Strategy Approval Templates

Reusable templates for strategy, approval, and executive communication.

## Navigation

| Template | Purpose | File |
|---|---|---|
| Strategy Doc | Define direction, goals, key bets, and priorities. | [strategy-doc.md](./strategy-doc.md) |
| Business Case | Justify the investment with problem, options, costs, benefits, risks, and ROI. | [business-case.md](./business-case.md) |
| Executive Summary | Give leaders a concise overview of the problem, proposal, and decision needed. | [executive_summary.md](./executive_summary.md) |
| Decision Memo | Present a recommendation and the rationale behind it. | [decision_memo.md](./decision_memo.md) |
| Executive Report | Summarize progress, KPI movement, risks, and key decisions for executives. | [executive_report.md](./executive_report.md) |
| Product Strategy One-Pager | Capture strategy on a single page for fast alignment. | [product_strategy_one_pager.md](./product_strategy_one_pager.md) |

## What’s inside

| File | What it is best for |
|---|---|
| `strategy-doc.md` | Long-term direction, strategic choices, and key bets. |
| `business-case.md` | Funding requests, ROI justification, and investment approval. |
| `executive-summary.md` | Fast executive readout and decision request. |
| `decision-memo.md` | Clear recommendation with tradeoffs and evidence. |
| `executive-report.md` | Periodic leadership reporting and KPI review. |
| `product-strategy-one-pager.md` | Quick strategic alignment in a compact format. |

## Suggested use order

1. Start with the `strategy-doc.md` when defining direction.
2. Use `business-case.md` when seeking approval or funding.
3. Use `executive-summary.md` when leadership needs a fast overview.
4. Use `decision-memo.md` when a decision must be made.
5. Use `product-strategy-one-pager.md` for quick alignment.
6. Use `executive-report.md` for recurring leadership updates.

## Notes

- Keep strategy documents concise and decision-oriented.
- Use examples to show expected depth and tone.
- Update the files when strategy or priorities change.
